# Sandena Umesh - Interactive 3D Portfolio 🚀

Welcome to my next-generation 3D personal portfolio! This project is built using modern web architecture to provide a premium, interactive experience that showcases my background in **Chemical Engineering** and my vision for **Sustainability & Innovation**.

![Portfolio Preview](https://community-filepreview.spline.design/webp-90/401875e7-6d8e-4027-9632-7a243433f3a1.webp)

## ✨ Features

*   **Interactive 3D Background**: Powered by **Spline**, featuring real-time mouse tracking that responds seamlessly to your cursor.
*   **Premium Glassmorphism UI**: High-end frosted glass aesthetics with smooth **Framer Motion** entry and scroll animations.
*   **Integrated AI Assistant**: A custom-built, embedded chatbot in the contact section designed to answer quick questions about my skills and vision.
*   **No-Build Architecture**: Optimized to run directly in the browser using React and Babel Standalone, making it extremely lightweight and portable.
*   **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.

## 🛠️ Tech Stack

*   **Core**: React 18
*   **3D Engine**: Spline Web Component
*   **Animations**: Framer Motion
*   **Styling**: Vanilla CSS (Custom Glassmorphism System)
*   **Compiler**: Babel Standalone (In-browser JSX transformation)

## 🚀 Getting Started

### Run Locally

Since this is a static site, you just need a local server to view it:

1.  Clone or download this repository.
2.  Open your terminal in the project folder.
3.  Run the following command (requires Python):
    ```bash
    python -m http.server 8000
    ```
4.  Open `http://localhost:8000` in your browser.

## 🌐 Deployment

This portfolio is ready for **GitHub Pages** or **Vercel** with zero configuration.

1.  **GitHub Pages**: Push this code to a GitHub repository, go to **Settings > Pages**, and enable deployment from the `main` branch.
2.  **Vercel**: Simply drag and drop the project folder into the Vercel dashboard.
